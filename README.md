# Todo CLI TUI

A terminal-based Todo CLI with keyboard-driven TUI.

## Running the Application

```bash
uv run python cli/main.py
```

## Usage

- Use **arrow keys** to navigate menu options
- Press **Enter** to select an option
- In task list, press **Enter** to return to menu

## Features (MVP)

- Add new tasks
- List all tasks
- Navigate with keyboard
- Clean exit
